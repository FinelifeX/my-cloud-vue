
# My Cloud Vue

Cloud system for working with files built using [Vue JS](https://vuejs.org/) and [TailwindCSS](https://tailwindcss.com/).

## Technologies used

- Webpack
- cross-env
- Babel
- ESLint (with eslint-config-airbnb-base)
- Prettier
- Vue JS (2.x)
- Vue Router
- VueX
- TailwindCSS
- SASS/SCSS
